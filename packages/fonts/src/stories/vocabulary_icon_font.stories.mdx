# Vocabulary Icon Font

Vocabulary Icons is an icon font developed for use in Vocabulary, a cohesive design system to unite the web-facing
Creative Commons.

The icon font selectively includes assets from [Font Awesome](https://fontawesome.com) based on the icons used
throughout the design library on Figma and bundles them using [IcoMoon](https://icomoon.io/) to get the icons in the
smallest possible file size bundle.

# Usage

To use the icons, add the classes `.icon`  and the name of the icon. Alternatively you can just use the ligatures by
typing out the icon name (without the dashes) inside an element with the `.icon` class applied. The standard CC icons
can also be used by typing out the Unicode character inside an element with the `.icon` class applied.

**Example:**

With classes

<IconGallery>
  <IconItem name="">
    <i class="icon chevron-up"></i>
  </IconItem>
</IconGallery>

```html
<i class="icon chevron-up"></i>
```

With ligatures

<IconGallery>
  <IconItem name="">
    <i class="icon">cclogo</i>
  </IconItem>
</IconGallery>

```html
<!-- Drop the dashes -->
<i class="icon">cclogo</i>
```

With Unicode characters (only standard CC icons)

<IconGallery>
  <IconItem name="">
    <i class="icon">ðŸ…¯</i>
  </IconItem>
</IconGallery>

```html
<!-- Your browser might not be able to display this -->
<i class="icon">ðŸ…¯</i>
```
