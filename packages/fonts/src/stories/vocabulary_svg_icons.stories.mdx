# Vocabulary Icons

export const iconItem = (name) => {
  return iconComponents[name]
}

Vocabulary Icons is an icon font developed for use in Vocabulary, a cohesive design system to unite the web-facing
Creative Commons.

The svg icons selectively include assets from [Font Awesome](https://fontawesome.com) based on the icons used
throughout the design library on Figma. They are optimized using [svgomg](https://jakearchibald.github.io/svgomg/) to
make the icons the smallest possible file size.

To use the icons, you have to have inline svg loader in your app. After it is set up,
copy the icon link and add it into your code:
```
import ccZero from '@creativecommons/vocabulary/assets/svg/cc-zero.svg?inline'
<span class='icon'><ccZero /></span>
```


export const createGallery = (icons) => {
  return (
    <IconGallery getCopyValue={iconLink}>
      { icons.map((i) => (
        <IconGallery.Item key={i} name={i} size={30}>
          {iconItem(i)()}
        </IconGallery.Item>
        ))
      }
    </IconGallery>
  )
}
