# Vocabulary Icons

Vocabulary Icons is an icon font developed for use in Vocabulary, a cohesive design system to unite the web-facing
Creative Commons.

The icon font selectively includes assets from [Font Awesome](https://fontawesome.com) based on the icons used
throughout the design library on Figma and bundles them using [IcoMoon](https://icomoon.io/) to get the icons in the
smallest possible file size bundle.

symbol replaces the pre-existing 'circled-equals' symbol (âŠœ / `&#8860;`) instead of occupying a new codepoint. They can
be copy-pasted into any location that is equipped to handle Unicode 13.0.
