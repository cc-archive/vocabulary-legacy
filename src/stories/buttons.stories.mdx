# Buttons

Buttons are created by adding the `.button` class to a `<button>`.

<Preview>
  <Story name="MediumButton" height="100px" parameters={{ design: figmaConfig('1403%3A61') }}>{`
    <button class="button">Button</button>
  `}</Story>
</Preview>

You can customize the size of buttons with the `.big`, `.small` and `.tiny` classes.

<Preview>
  <Story name="BigButton" height="100px" parameters={{ design: figmaConfig('1403%3A61') }}>{`
    <button class="button big">Button</button>
  `}</Story>
</Preview>

<Preview>
  <Story name="SmallButton" height="100px" parameters={{ design: figmaConfig('1403%3A61') }}>{`
    <button class="button small">Button</button>
  `}</Story>
</Preview>

<Preview>
  <Story name="TinyButton" height="100px" parameters={{ design: figmaConfig('1403%3A61') }}>{`
    <button class="button tiny">Button</button>
  `}</Story>
</Preview>

You can also combine other classes such as `.primary`, `.donate` and `.info` with the aforementioned size classes.

<Preview>
  <Story name="ButtonPrimary" height="100px">{`

  `}</Story>
</Preview>

<Preview>
