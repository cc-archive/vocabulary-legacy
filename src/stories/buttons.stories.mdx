# Buttons

Buttons are created by adding the `.button` class to a `<button>`.

<Preview>
  <Story name="MediumButton" height="100px">{`
    <button class="button">https://www.figma.com/file/l4Mt3dn3Ndtrvrb4aLcwXI/Design-Library?node-id=1403%3A61</button>
  `}</Story>
</Preview>

You can customize the size of buttons with the `.big`, `.small` and `.tiny` classes.

<Preview>
  <Story name="BigButton" height="100px">{`
    <button class="button big">https://www.figma.com/file/l4Mt3dn3Ndtrvrb4aLcwXI/Design-Library?node-id=1403%3A61</button>
  `}</Story>
</Preview>

<Preview>
