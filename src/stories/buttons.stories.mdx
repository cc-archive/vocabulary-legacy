# Buttons

export const buttonClass = (buttonClass) => {
 return `<button class="${buttonClass}" height="100px">Button</button>`
}

Buttons are created by adding the `.button` class to a `<button>`.

  <Story name="MediumButton" parameters={{

      ${buttonClass('button')}
  `}</Story>
</Preview>
