# Profile image

The profile image is a component to be used in every element that shows people
as single individuals or as a group. The size component can be used as it
needed, however, a `60px` square is a minimum size available.

You can use Bulma size helpers like `is-128x128` to set the image size.

<Preview>
  <Story name="Default" parameters={{
    design: figmaConfig('6728%3A329')
  }}>{`
    <figure class="image is-128x128">
      <img class="profile" src="demo/lion_dafrique.jpg">
    </figure>
  `}</Story>
</Preview>

You can also use CSS properties `height` and `width` to size the image.

<Preview>
  <Story name="Custom sizes" parameters={{
    design: figmaConfig('6728%3A329')
  }}>{`
    <style>
      .image {
        height: 256px;
        width: 256px;
      }
    </style>
    <figure class="image profile">
      <img class="profile" src="demo/lion_dafrique.jpg">
    </figure>
  `}</Story>
</Preview>
