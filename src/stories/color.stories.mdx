# Palettes

Creative Commonsâ€™ color palette reflects the vibrant nature of the digital space in which we operate, and the diverse
community we engage with. At the same time, the colors defined follow the accessibility indications made by W3C when
text is placed on them.

export const color = (name, hex, r, g, b) => `
  <style>
    .color {
      display: inline-block;
      margin: 0.5em;
    }
    .box-${hex} {
      border-radius: 0.25em;
      width: 10em;

  </style>
  <div class="color">
    <div class="box-${hex}">
    </div>
    <strong>${name}</strong><br/>
    #${hex}<br/>
    RBG(${r}, ${g}, ${b})
  </div>
`
