# License badge

License badges represent CC licenses using icons to depict clauses of the license. So the CC Attribution license or
CC BY is represented by the CC logo and the icon of a person. Similar icons exist for the NC, ND and SA clauses.

## Licenses

export const licenseSVG = (size, license) => {
  let tag
  if (size === 'big') {
    tag = 'svg height="42" width="120"'
  } else {
    tag = 'svg height="15" width="80"'
  }
  return `<${tag}>
<use href="license_badges/${size}/${license}.svg#layer1"></use>
</svg>`
}

# Usage

The license assets can be used in your apps by defining an SVG tag with the appropriate dimensions and linking to the
group from within the licenses.

| Size  | Dimension      |
|-------|----------------|
| Big   | 120 &times; 42 |
| Small | 80  &times; 15 |

**Example:**

```html
<svg width="120px" height="42px">
  <!-- Big sized CC BY license badge -->
  <use
    href="@creativecommons/vocabulary/assets/license_badges/big/by.svg#layer1">
  </use>
</svg>
<svg width="80px" height="15px">
  <!-- Small sized CC BY license badge -->
  <use
    href="@creativecommons/vocabulary/assets/license_badges/small/by.svg#layer1">
  </use>
</svg>
```
