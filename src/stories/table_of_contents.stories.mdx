# Table of Contents

The Table of Contents component shows the sections of a page and allows the user
to move across all sections on a page. It is recommended to fixed the component
while the user scrolls when the page is mainly for showing one type of content,
like a blog post.

The active section must change based on what the user is viewing on the content
section.

The component has two states, active and inactive. The hover state of inactive
elements should have the same style of active state.
